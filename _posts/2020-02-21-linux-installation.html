To install linux follow the below steps:

To start the installation from DVD/CD set the BIOS to boot from CDROM.
This installation guide is to install Linux without any other o/s present in the hard disk, where partition will be overwritten.

Installation can be done in two methods:
(1) GUI Mode
(2) Text Mode

Insert the first CD and restart the system
