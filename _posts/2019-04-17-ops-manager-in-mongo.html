Ops Manager is a monitoring and administering web based tool provided by Mongo which would provide a single click provisioning for scaling,upgrades and administering tasks.

Monitoring is done with charts , dashboards and alerts on 100+ metrics.Backup and restore with point in time recovery , support for sharded clusters is also provided.

Visual query performance diagnostics , index suggestion and automated index rollouts are also provided.

Deployement

Deploy and manage any type of Mongo DB implementation
-Replica Set
-Sharded Environments
-Standalone Server

Designed to support a wide variety of architecture:
- Public Cloud
-Private Cloud
-Bare Metal

Also it allows for detailed parameter tuning.


Few operations which can be performed in Ops Manager are:
* Version upgrades and downgrades
*Resizing Oplog
*Change startup parameters
*Build indexes
*Change listening ports/Storage Engine
*Enable Profiler
*Tune Storage Engine Parameters
*Change member priority/vote/replication delay
